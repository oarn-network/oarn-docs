# OARN Discord Server Setup Guide

## Server Creation

2. Click "+" to create a new server

4. Name: **OARN Network**

## Channel Structure

### Information
- `#welcome` - Server rules and intro
- `#announcements` - Official announcements (admin only)
- `#roadmap` - Project milestones and updates

### Community
- `#general` - General discussion
- `#introductions` - New member introductions
- `#ideas` - Feature suggestions

### Development
- `#dev-general` - Development discussion
- `#smart-contracts` - Solidity/contract discussions
- `#node-software` - Rust node discussions
- `#infrastructure` - Infrastructure and DevOps
- `#code-review` - PR and code review requests

### Node Operators
- `#node-setup` - Help with running nodes
- `#node-troubleshooting` - Technical issues
- `#hardware` - Hardware recommendations

### Research
- `#research-general` - Research discussions
- `#papers` - Academic paper discussions
- `#partnerships` - Research partnership discussions

### Governance
- `#governance` - DAO and voting discussions
- `#proposals` - Proposal discussions

### Voice
- `Voice Chat` - General voice
- `Dev Standup` - Development meetings

## Roles

| Role | Color | Permissions |
|------|-------|-------------|
| Core Team | Red | Admin |
| Moderator | Orange | Manage messages |
| Contributor | Blue | Post in dev channels |
| Node Operator | Green | Access to operator channels |
| Researcher | Purple | Access to research channels |
| Community | Gray | Basic access |

## Verification

Enable phone/email verification for new members to prevent spam.

## Server Settings

- Community server: ON
- Rules screening: ON
- Membership screening: ON
- 2FA for moderators: ON
